보안취약점 및 개인정보보호 체크리스트
소스 보안취약점 및 개인정보보호 상세 체크리스트

1.  입력 데이터 검증 및 표현
  - SQL 인젝션 방지: MyBatis/JPA 사용 시 파라미터 바인딩 사용 여부
  - XSS: HTML 엔티티 치환 처리 여부
  - 경로 조작: 외부 입력값의 파일 경로 직접 사용 금지
  - 파일 업로드: 실행 파일 제한 및 서버 사이드 체크

2.  보안 기능
  - 취약 암호화 알고리즘 금지 (AES-256 이상 권장)
  - 하드코딩 금지 (DB 정보, API 키 등)
  - 중요정보 HTTPS 전송

3.  개인정보보호
  - 화면 마스킹 처리
  - 로그 내 민감정보 노출 금지
  - 고유식별정보 DB 암호화
  - 다운로드 사유 기록

4.  에러 처리 및 기타
  - 시스템 정보 노출 금지 (Stack Trace)
  - 디버그 코드 제거 (System.out.println 등)
